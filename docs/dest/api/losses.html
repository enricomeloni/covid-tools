class lcm.losses.target_losses.RMSE(weights, **kwargs)¶

Root Mean Square Error.

target_loss(hat, target, mask)¶

Computes the RMSE loss on target values greater or equal than 0.

Parameters

  • hat – A tensor of predictions.

  • target – A tensor with targets.

  • mask – An tensor for masking elements

Returns

A scalar value (still tensor) corresponding to the RMSE between inferences and targets. Values of the target lower or equal than 0 are NOT accounted in the computation.
